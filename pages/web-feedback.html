Help us to make the website better

We really want to make sure this website is a friendly and useful resource for researchers, students, and others who want to learn about reproducibility. Please help us to improve it in one of the following ways:

Complete the form

Create an issue in Github

The website is hosted on Github. If you are familiar with Github and have suggestions about improving the website, please create an issue on our repository.

Create a Pull Request in Github

If you are familiar with Github and found a broken link, typos, or other errors that can be fixed without too much discussion, please create a pull request on our repository.
